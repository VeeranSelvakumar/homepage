Learning License Apply செய்ய தேவையான ஆவணங்கள்

Learning License (LLR) விண்ணப்பத்தை எளிதாக சமர்ப்பிக்க கீழே கொடுக்கப்பட்டுள்ள ஆவணங்களை தயாராக வைத்திருக்கவும்.

📄 தேவையான ஆவணங்கள்

  • ✅ ஆதார் கார்டு (உங்கள் ஆதார் கார்டில் பெயர், பிறந்த தேதி , முகவரி மற்றும் மற்ற விவரங்கள் அனைத்தும் எந்த பிழைகளும் இல்லாமல் இருக்கிறதா என்பதை உறுதி செய்து கொள்ளவும்)
  • ✅ ஆதார் கார்டுடன் இணைக்கப்பட்ட Mobile Number மற்றும் அந்த எண் ஆக்டிவாக இருக்க வேண்டும்
  • ✅ Passport Size Colour Photo ( தேவைப்பட்டால் )
  • ✅ Signature

A: Aadhaar Authentication தேர்வு செய்தால் Aadhaar மற்றும் Mobile Number கட்டாயம்.

Q: Passport Size Photo தேவையா?

A: ஆம். சமீபத்திய வண்ண புகைப்படம் தேவைப்படும்.

Q: Signature Upload செய்ய வேண்டுமா?

A: ஆம். வெள்ளை தாளில் கையொப்பமிட்டு Upload செய்ய வேண்டும்.

Q: Learning License பெற குறைந்தபட்ச வயது எவ்வளவு?

A: பொதுவாக 16 அல்லது 18 வயது (வாகன வகையைப் பொறுத்து).
