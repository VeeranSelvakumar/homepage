Eligibility

Minimum age

01.07.2026 andru minimum age 21 years irukkanum.

Maximum age

Post-wise age limit maarum. 32, 34, 35, 37 maadhiri upper age limits mention pannirukku.

Age relaxation

BC, BCM, MBC/DNC, SC, SCA, ST categories-ku pala posts-la upper age limit illa-nu mention pannirukku.

"Upper age limit illa" nu sonnalum recruitment process mudiyum nerathula 60 vayathai kadandhirukka koodadhu.

Age proof

  • SSLC marksheet
  • HSC marksheet
  • Birth certificate
  • Transfer certificate
  • Degree marksheet

Age proof upload pannaama vittaal application reject / selection cancel aagalam.

Degree-wise main eligible posts

  • Civil: Assistant Engineer posts (Highways, Water Resources, Rural Development, Industries), Assistant Manager (Civil), Planning related posts.
  • Mechanical: Assistant Engineer (Mechanical), Boiler Junior Assistant Director, some mechanical department posts, secondary eligibility for Agricultural Engineering posts in certain cases.
  • Electrical / EEE: Assistant Engineer (Electrical), Assistant Manager (Electrical).
  • CSE / IT / ECE: Computer Manager, Assistant Manager (IT).
  • Agricultural Engineering: Assistant Engineer (Agricultural Engineering); primary preference Agricultural Engineering, then certain secondary branches if primary candidates illa.
  • B.Sc Agriculture: Agricultural Officer, some warehouse/junior assistant related eligibility; Tamil knowledge compulsory.
  • B.Sc Horticulture: Horticultural Officer only.
  • Geology: Assistant Geologist related posts.
  • Library Science: Librarian, Assistant Librarian.
  • Law: Translator.
  • Archaeology / History: Archaeological Officer, Assistant Archaeological Officer.
  • Pharmacy / Medical related: Drug Inspector.
  • Statistics / Maths / Economics: Block Health Statistician, Assistant Statistical Investigator.
  • MBA: Regional Manager, Assistant General Manager.
  • Visual Communication: Junior Photographer.
  • Dairy / Food Tech: Deputy Manager (Dairy Farm).

Exam Pattern & Selection

Paper-I rule

  • Minimum 40% marks edukkanum.
  • Paper-I qualify pannaama irundha Paper-II evaluate panna maattaanga.
  • Tamil paper mark final rank-ku count aagathu.

Paper-II rule

  • Idhu main competitive paper.
  • Final ranking Paper-II marks basis-la thaan.
  • Subject paper degree-ku match aaganum.

Selection flow

Application → Hall Ticket → Written Exam → Tamil Paper Qualify → Paper-II Evaluation → Rank List → Certificate Verification → Final Selection

  • Destitute Widow Certificate (if applicable)
  • Experience Certificate (if required)

Reservation & Category Rules

Main categories

  • BC
  • BCM
  • MBC/DNC
  • SC
  • SCA
  • ST
  • OC
OC na general category-nu artham illa. High mark eduthaal BC / MBC / SC / ST candidate kooda OC seat-la varalaam.

PSTM & special categories

  • PSTM claim panna required qualification muzhuvadhum Tamil medium-la padichirukkanum.
  • PSTM certificate government format-la upload pannanu.
  • Differently abled reservation ella posts-kum illa; notification post-wise paakanum.
  • Ex-servicemen, Destitute Widow-ku special benefits / relaxation irukkum.

Important reservation points

  • Final selection = merit + reservation + category + communal turn.
  • Category-wise cut-off separate-a vary aagum.
  • Same mark vandha DOB seniority, reservation rules, communal turn, special priority paaka chance irukku.
  • Community, PSTM, Disability, Ex-servicemen, Destitute Widow claims-ku original proof mandatory.

Application Process

Basic steps

  • First time applicant na OTR (One Time Registration) seiyanum.
  • OTR pannina mattum application complete aagathu.
  • OTR + individual application rendu perum complete pannanum.

OTR details

  • Name
  • Date of Birth
  • Aadhaar Number
  • Mobile Number
  • Email ID
  • Community Details
  • Educational Details

Fees

  • One Time Registration Fee: ₹150
  • Examination fee notification basis-la apply aagum.
  • SC, SCA, ST, PwBD, Destitute Widow categories-ku fee concession / exemption irukkalaam.

Upload rules

  • Photo: recent passport size, clear face, white/light background.
  • Signature: own signature, black/blue ink, white paper.
  • Certificates: education, reservation, experience documents clear-a upload pannanum.

Hall ticket & exam day

  • Hall ticket-la name, register number, exam date, time, centre, photo verify pannanum.
  • Carry: Hall Ticket + valid ID proof (Aadhaar / Voter ID / Passport / Driving Licence).
  • 1 mani neram munnadi exam centre reach pannradhu nalladhu.

Exam hall-la carry panna koodaadha items

  • Mobile phone
  • Smart watch
  • Bluetooth devices
  • Calculator
  • Books
  • Notes
  • Electronic gadgets

Most Common Mistakes

Application / eligibility mistakes

  • Degree mismatch
  • Equivalent degree proof illaamal apply pannuvidudhal
  • Tamil qualification illaama Agricultural Officer-ku apply seivadhu
  • Additional diploma / registration thevaiyana posts-ku proof illaamal apply seivadhu
  • Wrong subject paper selection
  • Wrong post code selection
  • Wrong community / category entry

Document / technical mistakes

  • Age proof upload pannaamal விடுதல்
  • PSTM claim without certificate
  • Blurred scan / incomplete scan / old certificate upload
  • Name mismatch between records
  • Photo mismatch / signature mismatch
  • Hall ticket details verify pannaamal விடுதல்
  • False information declaration

Top reasons for losing selection

  1. Tamil Eligibility Test fail
  2. Wrong community entry
  3. Degree mismatch / wrong degree entry
  4. Missing certificate
  5. Wrong subject paper
  6. Age proof missing
  7. PSTM certificate missing
  8. Hall ticket details not verified
  9. Document verification failure
  10. False information

Final Checklist

Before applying

  • Ungal degree-ku exact eligible posts shortlist pannunga.
  • Age limit, community benefit, PSTM eligibility verify pannunga.
  • Correct subject paper and post code choose pannunga.

Before submit

  • Name, DOB, community, degree details verify pannunga.
  • All uploaded documents clear-a irukka paathukkonga.
  • Required certificates miss aagala-nu final check pannunga.

Before CV / exam

  • Original + xerox copies ready-a vainga.
  • Hall ticket + valid ID carry pannunga.
  • Tamil paper-ai ignore pannaadheenga; 40 marks mandatory.
